What is the Wildy Minecraft Server?

The AxleCraft.com Wildy Minecraft Server is a members-only survival server where the entire world is a dangerous wilderness except a few sanctuary zones. You can choose to play solo and build a base far away to reduce the chance your base is found and looted, or you can form alliances with other players and cooperate to construct epic team bases. Members can choose whether they want to siege enemy fortresses, or live in harmony to create a tranquil utopia.

How to Join and Play on the Server

AxleCraft Wildy is a private whitelisted version 1.8 minecraft server. Members can join by donating $40 for server lifetime access, or donating $10 for 1 month of access. Most crypto coins are accepted. If you are interested in playing send an application to AxleCraft#2110 on Discord. Everyone can get a free 2 week trial by completing the first active bounty.

Your application must include the following information:

  • Your email address
  • Your minecraft username
  • Your youtube username
  • The name and amount of coin that you want to send
  • The wallet address you want to send from
  • A statement saying youve read the rules and agree to them

Server Rules

In general there are no rules for gameplay, the rules are up to each individual member gamer since this is a wilderness server. Admins reserve the right to ban. If you feel your ban was unjust send an appeal

Warning: A ban and loss of membership will occur if you violate any of the the following:

  • No hacks, exploits, duping, or modded clients are allowed. Anything that gives you an advantage is banned such as x-ray vision or auto diamond mining. If you are detected using a modded client or it is obvious that you are and other players report you, you will be banned
  • Dont use obsidian or other similar blocks to obstruct spawn point (0,0), or to trap other players who cant escape from it
  • Dont destroy the marked walled sanctuary zones, main highways or nether highways (+/- x, +/- z)
  • All names, language, and conduct must remain PG-rated since this server may be streamed
  • Dont have any expectations of work from AxleCraft or admins, or refunds of donations

If your application is missing data it will be denied. This application filtering is intended to create a high quality dedicated group of members on this server. You can use https://privnote.com/ to send your application data over discord. Please dont send any coins until your application has been approved by AxleCraft unless you just want to donate to help the server, then thank you! If your application is approved, you can choose to donate to the wallet address provided. This contribution is a donation to AxleCraft and it can allow you to be whitelisted to join the minecraft server.

Server Lifetime

The server will run perpetually if members support it. The server will run for atleast 1 year (January 2022 to the end of December 2022), however the server will continue to remain live if a few community members support it each month. Devs might upgrade the server code and infrastructure with continued member support.

Most Coins Accepted

Most liquid coins and chains are supported. When you send in your application AxleCraft will notify you if the coin you want to use is not accepted before you send anything.
